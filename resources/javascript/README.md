# JavaScript — resources

> Curated external material for JavaScript the language. My own explainers are in [../../references/javascript/](../../references/javascript/); the runtime lives in the `node` domain and types in `typescript`.

## Official docs & specs

- **[MDN JavaScript](https://developer.mozilla.org/en-US/docs/Web/JavaScript)** — the practical reference for the language and standard library; the JavaScript Guide is the best free long-form intro.
- **[TC39 proposals](https://github.com/tc39/proposals)** — what's coming to the language and at which stage; the "finished proposals" list is the canonical "what shipped."
- **[ECMA-262 (the spec)](https://tc39.es/ecma262/)** — the authoritative language definition. Dense, but the final word when behavior is disputed.

## Learn

- **[javascript.info](https://javascript.info/)** — "The Modern JavaScript Tutorial": thorough, current, well-sequenced from basics to advanced.
- **[Eloquent JavaScript](https://eloquentjavascript.net/)** — free book, strong on programming fundamentals with JS.
- **[You Don't Know JS Yet (2nd ed.)](https://github.com/getify/You-Dont-Know-JS)** — deep dives on scope/closures, `this`, types/coercion — exactly the tricky areas.

## Cheatsheets & quick-reference

- **[MDN — Equality comparisons and sameness](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Equality_comparisons_and_sameness)** — the `==`/`===`/`Object.is` table.
- **[caniuse](https://caniuse.com/)** · **[node.green](https://node.green/)** — which JS features work in which browsers / Node versions.

## Tools & utilities

- **[ESLint](https://eslint.org/)** · **[Prettier](https://prettier.io/)** — lint and format (the dedicated tooling domain covers configs).
- **[Babel REPL](https://babeljs.io/repl)** — see how modern syntax compiles; useful for understanding sugar like classes/async.
- **[AST Explorer](https://astexplorer.net/)** — inspect the parse tree; helpful for writing lint rules / codemods.

## Practice

- **[Exercism — JavaScript track](https://exercism.org/tracks/javascript)** — exercises with mentoring.
- **[JavaScript questions (lydiahallie)](https://github.com/lydiahallie/javascript-questions)** — annotated quiz that targets coercion/`this`/closures edge cases.
