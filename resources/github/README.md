# GitHub — resources

> Curated external material for the GitHub platform. My own explainers (PRs, Actions, `gh`, rulesets, security) are in [../../references/github/](../../references/github/); this catalog points outward. Git-itself resources are in [../git/](../git/).

## Official docs & specs

- **[GitHub Docs](https://docs.github.com/)** — the authoritative documentation hub for every GitHub feature.
- **[GitHub Actions docs](https://docs.github.com/en/actions)** — workflows, events, contexts, and the full workflow syntax reference.
- **[`gh` CLI manual](https://cli.github.com/manual/)** — every `gh` command and flag.
- **[REST API reference](https://docs.github.com/en/rest)** · **[GraphQL API](https://docs.github.com/en/graphql)** — what `gh api` calls under the hood.
- **[Workflow syntax for GitHub Actions](https://docs.github.com/en/actions/using-workflows/workflow-syntax-for-github-actions)** — the precise YAML schema for workflows.

## Learn

- **[GitHub Skills](https://skills.github.com/)** — short, hands-on courses run inside real repos (PRs, Actions, pages).
- **[Actions: quickstart](https://docs.github.com/en/actions/quickstart)** — the fastest path to a first working workflow.

## Cheatsheets & quick-reference

- **[GitHub CLI cheat sheet (manual examples)](https://cli.github.com/manual/examples)** — copy-paste `gh` usage for common tasks.
- **[Mastering Markdown](https://docs.github.com/en/get-started/writing-on-github)** — GitHub-flavored Markdown reference (tables, task lists, alerts, mermaid).

## Tools & utilities

- **[Actions Marketplace](https://github.com/marketplace?type=actions)** — published, reusable actions to drop into a workflow with `uses:`.
- **[`act`](https://github.com/nektos/act)** — run GitHub Actions workflows locally for faster iteration than push-and-wait.
- **[GitHub Status](https://www.githubstatus.com/)** — check before assuming a push/CI failure is your fault.

## Security

- **[GitHub Advisory Database](https://github.com/advisories)** — the vulnerability data behind Dependabot alerts.
- **[Security hardening for GitHub Actions](https://docs.github.com/en/actions/security-guides/security-hardening-for-github-actions)** — token scoping, pinning actions, untrusted-PR handling.
